# 📝 Gerenciador de Tarefas Acadêmicas - Next.js 15 & Jest

Este projeto é uma aplicação prática desenvolvida para consolidar os conceitos de **testes unitários** utilizando o **Next.js 15 (App Router)**, TypeScript, **Jest** e **React Testing Library**.

## 🚀 Funcionalidades Solicitadas no Desafio
- Exibição de uma lista de tarefas simulando uma resposta assíncrona do servidor (Server Component).
- Adição de novas tarefas através de um formulário controlado (Client Component).
- Contagem dinâmica do total de tarefas utilizando um Hook Personalizado isolado.
- Cobertura completa de testes unitários para todos os fluxos críticos.

## 🛠️ Tecnologias Utilizadas
- **Next.js 15** (com App Router)
- **React 19**
- **TypeScript**
- **Jest** & **React Testing Library**
- **ts-jest** (para execução nativa de testes em arquivos `.ts/.tsx`)
