# Gerenciador de Tarefas - Next.js & Jest

Uma aplicação web robusta desenvolvida com **Next.js**, **TypeScript** e **React 19** para listagem e adição de tarefas. O projeto foi estruturado seguindo boas práticas de arquitetura de software, separando componentes de servidor (Server Components), componentes de cliente (Client Components), hooks customizados e testes automatizados.

---

## Tecnologias Utilizadas

* **Framework:** Next.js (App Router)
* **Linguagem:** TypeScript
* **Biblioteca Base:** React 19
* **Testes:** Jest & React Testing Library
* **Compilação de Testes:** Babel
